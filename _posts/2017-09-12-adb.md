## 备份数据

```
adb backup -apk -shared -all -f 2017-0912-backup.db
```

## 恢复数据

```
adb restore 2017-0912-backup.db
```

## 复制文件到电脑

```
adb pull /sdcard/data.tar.gz ./
```

## 复制文件到手机

```
adb push data.tar.gz /sdcard/
```
